# Navi

Navi is designed for avatar caching, with style. 

The name of **Navi** from movie [Avatar](https://en.wikipedia.org/wiki/Avatar_(2009_film)).

## Requirements

Swift 2.0, iOS 8.0

## Example

Check the demo for more information.

## Installation

### CocoaPods

```ruby
pod 'Navi', '~> 0.3'
```

### Carthage

```ogdl
github "nixzhu/Navi" >= 0.3
```
